## 1. 设计原则与视觉语言

**定位**：Apple 级质感 × Bloomberg 级信息密度。前台商城偏"奢侈品电商"（留白、大图、克制），后台工作台偏"专业终端"（高密度表格、多面板、键盘友好）。

### 1.1 设计令牌（Design Tokens，Tailwind + shadcn 主题）

| 令牌 | Light | Dark | 用途 |
|---|---|---|---|
| `--background` | #FAFAF8 暖白 | #0C0C0E | 页面底 |
| `--foreground` | #1A1A1A | #EDEDEA | 正文 |
| `--primary` | #0F1B2D 深海军蓝 | #E8E6E0 | 主按钮/品牌 |
| `--accent` | #B08D46 鱼子酱金 | #C9A55C | 强调/等级徽章/CTA hover |
| `--muted` | #6B6B66 | #8A8A85 | 次要文本 |
| `--destructive` | #B3261E | #F2B8B5 | 危险操作 |
| `--success / --warning` | #1D7A5F / #9A6B0F | 对应亮色 | 状态 |
| 圆角 | `--radius: 6px`（后台 4px 更硬朗） | | |
| 字体 | Inter（拉丁）+ Noto Sans SC（中文）；数字表格用 tabular-nums | | |
| 字号阶 | 12/13/14(基准)/16/20/24/32 | | 后台基准 13px 提密度 |

- **主题**：Light/Dark 双模式（`next-themes`，跟随系统 + 手动切换）；所有颜色经由 CSS 变量，禁止硬编码。
- **等级视觉**：G001–G004 等级徽章用金/铂/银渐进的**描边**徽章（非彩色块），奢侈品克制感。
- **图标**：lucide-react（shadcn 默认），线性风格统一。

### 1.2 组件规范（packages/ui，shadcn 封装）

基础：Button/Input/Select/Dialog/Sheet/Tabs/Toast/Tooltip/Badge/Skeleton（全部 shadcn）。
业务组件（自建，全端复用）：

| 组件 | 说明 |
|---|---|
| `<PartyCode>` | 平台代码徽章（SP-000018），hover 显示认证状态；Broker 视图附"申请穿透"入口 |
| `<PriceTierTable>` | 阶梯价表（币种切换、当前档高亮） |
| `<StatusPill>` | 状态机状态药丸（颜色映射数据化，与状态机定义同源） |
| `<TraceTimeline>` | 溯源时间线（脱敏级别由后端裁剪，前端仅渲染） |
| `<MoneyText>` | 金额（币种符号、千分位、tabular-nums、汇率快照 tooltip） |
| `<LocaleText>` | 翻译内容渲染（带"机器翻译"角标当 status=MACHINE_DRAFT） |
| `<ScoreRadar>` | 商机四维评分雷达/条形 |
| `<DocChecklist>` | 单证齐备度清单（缺件红点） |
| `<ColdChainChart>` | 温度曲线（超阈值区间标红） |
| `<CountdownPill>` | 倒计时（拍卖/意向单 24h/争议期） |

### 1.3 国际化与响应式

- 语言切换器全局常驻（顶栏）；URL 前缀 `/zh /en /fr`；法语文本平均比中文长 40%——所有按钮/标签禁止固定宽度。
- 断点：前台商城完整响应式（移动优先）；工作台 ≥1280px 优化（桌面为主），<768px 降级为只读关键信息 + 通知。
- 无障碍：WCAG 2.1 AA；表格键盘导航；焦点环不可移除。

## 3. 关键页面设计

### 3.1 市场目录（/market）

```
┌ 顶栏：Logo | 市场 拍卖 关于 | ⌘K 搜索 | 语言 | 主题 | 登录 ┐
├ 筛选栏(左,240px)         │ 结果区                          │
│  品类树                  │  排序: 最新/价格 | 视图: 卡片/表格 │
│  品种(多选)              │  ┌────┐ ┌────┐ ┌────┐          │
│  等级 | 规格 | 产地(国家) │  │产品图 │ │    │ │    │  3-4列  │
│  价格区间(登录可见)       │  │名称·等级徽章·SP-xxxx          │
│                          │  │€800/kg 起(登录) · 库存:充足   │
└──────────────────────────┴────────────────────────────────┘
```
要点：供应商只显示 `<PartyCode>`；库存只显示档位；未登录价格区显示"登录查看批发价"引导注册（获客漏斗）。

### 3.2 产品详情（/market/:code）

上半屏：左图集（大图 + 缩略）、右侧信息卡（名称三语、等级徽章 + 等级标准弹层、品种拉丁名、`<PriceTierTable>`、MOQ、`<TraceTimeline>` 摘要、认证徽章行、CTA：加入购物车 / 发起 RFQ / 咨询（IM））。
下半屏 Tabs：描述 ｜ 溯源详情（脱敏）｜ 规格参数 ｜ 合规单证（仅认证状态徽章，不给原件）。

### 3.3 Broker 商机看板（平台核心屏，Bloomberg 密度）

```
┌ 概览条: 今日新商机 12 | 跟进中 8 | 本月成单 €86k | 我的佣金贡献 ┐
├ 商机流(左,主区)                      │ 右侧面板(选中商机)      │
│ ▼ 按 Urgency 排序 · 筛选[状态|信号]  │ ┌ BY-000256 (脱敏360)  │
│ ┌──────────────────────────┐        │ │ 类型:米其林餐厅·FR    │
│ │ 🔴 OPP-0142  匹配 95      │        │ │ 偏好: DAU 100g · 月采 │
│ │ BY-000256 ↔ SP-000018    │        │ │ 30天未下单 ⚠         │
│ │ 史氏鲟鱼子酱 50g × 50kg   │        │ ├ 操作区               │
│ │ U:92 P:88 · 信号:断货周期 │        │ │ [📞外呼] [✉代发消息]  │
│ └──────────────────────────┘        │ │ [📄脱敏发单] [🤝代下单]│
│ │ OPP-0141 ...              │        │ │ AI 建议: 开场白/可让价 │
└──────────────────────────────────────┴────────────────────────┘
```
要点：新商机 WS 红点 + 声音可选；每张卡显示四维分 `<ScoreRadar>` 缩略与评分解释 tooltip；操作全部在右侧面板完成（不离屏）；外呼后状态条实时显示（拨号中/通话 03:12）；代下单弹出向导（选批次 → 数量价格 → 定金比例 → 预览 Buyer 收到的支付通知）。

### 3.4 供应商发货向导（体现单证守卫）

Stepper：① 分配批次（从可用 Lot 选，自动校验数量/效期）→ ② 物流段录入（多段 Leg）→ ③ 单证核对（`<DocChecklist>` 7 件套，缺件红色禁止下一步，每件可上传/关联已有单证）→ ④ 确认发货。任何一步保存草稿。

### 3.5 订单详情（Buyer 视角）

左侧主列：状态机时间线（每个节点带时间戳与操作方"平台/供应商 SP-xxx"）→ 物流卡（多段 Leg + `<ColdChainChart>`）→ 单证下载区（脱敏副本，带水印）。右侧：金额卡（明细/佣金透明度按策略）、支付记录、发票下载、"发起争议"入口（争议期倒计时 `<CountdownPill>`）。

### 3.6 审计中心：穿透审批

超管队列表格：申请人/目标/字段/事由/敏感级/等待时长；行内 通过/拒绝；通过后显示剩余窗口倒计时。全局"穿透热力"小卡：本周穿透次数 Top 客服（风控信号）。

### 3.7 IM 会话（拦截体验）

消息输入框实时检测：命中电话/邮箱时输入框边框变红 + 内联提示"平台规则：请勿交换联系方式（第 2/3 次警告）"，发送被服务端 422 拒绝时消息显示"已拦截"灰条（仅发送者可见）。会话头部显示挂靠业务对象（订单/产品卡片）。

## 5. 前端技术规范

- App Router + RSC：前台目录/详情用 Server Components（SEO/首屏）；工作台交互区 Client Components + TanStack Query（缓存失效按资源 tag）。
- 表单：react-hook-form + zod（与后端同 schema，`packages/shared`）。
- 表格：TanStack Table（虚拟滚动，后台大列表）。
- 状态：服务端状态归 Query；本地 UI 状态 zustand（购物车乐观更新）。
- WS：socket.io-client 单连接多命名空间，断线重连 + 事件重放（`lastEventId`）。
- 性能预算：前台 LCP < 2.5s（图片 next/image + OVH CDN）、工作台首屏 JS < 300KB gz。
